# Meta XR minimal sample

This Unity 6000.3 sample contains an `OVRCameraRig`, Touch controller grabbers, and one
`OVRGrabbable` cube. Run `MetalQuestLink.Sample.Editor.SampleBuilder.ConfigureAndBuild` once to
generate the scene and configure Standalone OpenXR, then select Meta XR Simulator and press Play.

The local `com.metalquestlink.editor` dependency registers the native layer and prepares the Quest
connection automatically.
