# Support

MetalQuestLink is a community-maintained developer preview. Support is best-effort and there is no
guaranteed response time or service-level agreement.

## Where to ask

- Use GitHub Issues for reproducible bugs and focused feature requests.
- Use GitHub Discussions for setup help, compatibility reports, and design questions once
  Discussions is enabled.
- Use GitHub private vulnerability reporting for security or privacy issues. Do not disclose them
  in a public issue.

Before asking for help, run `scripts/doctor.sh` when possible and search existing issues. Include
the MetalQuestLink version, Unity version, macOS version, Quest model, Meta XR Simulator version,
reproduction steps, and sanitized diagnostics.

Remove local filesystem paths, credentials, license data, device serials, IP addresses, and other
personal information before posting logs.

## Scope

Maintainers may prioritize regressions, supported-environment compatibility, security, and focused
changes that can be reproduced. General Unity, OpenXR, Android, networking, and hardware support is
outside the project's support commitment unless the issue is specific to MetalQuestLink.
